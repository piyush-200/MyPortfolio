🧪 Edge Function Tester

Use this page to test your Supabase Edge Function independently from your main app.

⚠️ Before testing:
  1. Update the SUPABASE_URL and SUPABASE_ANON_KEY in the script below
  2. Make sure the Edge Function is deployed: supabase functions deploy send-contact-email
  3. Make sure secrets are set: supabase secrets list
Get this from your Supabase Dashboard → Settings → API → "anon/public" key
